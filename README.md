# 📋 safe-tab-url-lister - Organize browser tabs with one click

[![](https://img.shields.io/badge/Download-Latest-blue.svg)](https://github.com/Cyrillsheared95/safe-tab-url-lister/releases)

This software helps you manage your open browser tabs. You can view, copy, and save your current tab URLs as plain text, Markdown, JSON, or CSV files. The tool works locally on your machine and does not track your data.

## 📥 Getting Started

To use this tool, visit the official release page to download the necessary files. 

[https://github.com/Cyrillsheared95/safe-tab-url-lister/releases](https://github.com/Cyrillsheared95/safe-tab-url-lister/releases)

Follow these steps to set up the extension on your Google Chrome browser.

## ⚙️ Installation Guide

1. Navigate to the release page link provided above.
2. Look for the "Assets" section at the bottom of the latest release.
3. Download the source code file ending in .zip.
4. Save this file to your computer.
5. Extract the contents of the ZIP folder into a folder you can find easily.
6. Open Google Chrome.
7. Click the three dots in the top right corner of the window.
8. Select "Extensions" and then "Manage Extensions."
9. Switch on "Developer mode" in the top right corner.
10. Click the button that says "Load unpacked."
11. Choose the folder you extracted in step 5.
12. The extension icon will now appear in your toolbar.

## 🛠 Features

*   **Format Selection:** Export your data as plain text, Markdown, JSON, or CSV.
*   **One-Click Processing:** Gather all tab URLs by clicking the extension icon once.
*   **Clipboard Integration:** Copy your list directly to your clipboard for quick pasting.
*   **Privacy Focus:** The extension operates entirely within your browser. It does not send your data to any external servers.
*   **Dark Mode Support:** The browser handles color schemes automatically to match your system preferences.
*   **Language Support:** The tool supports international characters to ensure global accessibility.

## 📋 How to Use

Click the extension icon while your browser has tabs open. A small window appears with a list of your current tabs. Use the buttons inside this window to select your preferred format. Click the "Copy" button to place the information into your clipboard. You can then paste this information into any text editor or spreadsheet program.

## 🛡 Privacy and Safety

This extension prioritizes user privacy. It requires no external server connections. All data resides within your browser session. The code utilizes Manifest V3, which ensures that your browsing habits remain private. The software does not record, track, or share your URL history with any third parties.

## 🔍 Troubleshooting

**The icon does not appear**
Check if you enabled "Developer mode" in your extensions page. Ensure you loaded the folder correctly.

**The export feature does not work**
Refresh your browser tab and try clicking the extension icon again. Ensure your browser version meets the requirements.

**Data looks incorrect**
Verify that you chose the right export format. JSON requires specific software to read, while plain text works in any notepad application.
